# Datasets

This project uses publicly available datasets for the following threat detection tasks:

- Fall Detection
- Fire Detection
- Violence Detection

The datasets are **not included** in this repository due to GitHub storage limitations.

## Folder Structure

```text
Dataset/
├── Fall Dataset/
├── Fire Dataset/
└── Violence Dataset/
```

## Instructions

Download the required datasets from their respective public sources and place them in the folders shown above before running the project.

> **Note:** The dataset folder names should remain unchanged to ensure the project runs correctly.
